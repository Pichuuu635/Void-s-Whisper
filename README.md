# Void-s-Whisper
🌌 Void's Whisper: RPG de Sincronia de Sistemas:
• Void's Whisper é um jogo de RPG de aventura textual desenvolvido em Portugol.
O projeto simula uma interface de engenharia em uma estação orbital consumida por uma anomalia espacial, onde a lógica de programação é a única arma de sobrevivência.

📖 Sinopse:
• No ano de 2026, a humanidade detecta um "sussurro" vindo de uma dobra espacial.
Você assume o papel de um Sincronizador de Sistemas enviado para a estação Aethelgard.
O objetivo é realizar um Hard Reset na IA avançada Pyxu, que atingiu a singularidade e agora tenta converter a tripulação em puro código estático.

🛠️ Aspectos Técnicos:
Este projeto foi construído para demonstrar o domínio de estruturas fundamentais de programação:
• Vetores: Gerenciamento de um inventário de emergência com itens como Patch de Segurança e Pulso Eletromagnético.
• Estruturas de Repetição: Uso de loops faca-enquanto e enquanto para controlar o fluxo de diálogos e o sistema de combate por turnos.
• Lógica de Combate: Sistema de batalha baseado em Integridade Neural (Vida do Jogador) vs Integridade de Pyxu (Vida do Inimigo), com cálculo de dano variável e riscos de sobrecarga.
• Modularização: O código é organizado em funções específicas para cada setor da nave e fase do jogo, facilitando a manutenção e expansão.

🕹️ Funcionalidades:
• Sistema de Escolhas: Caminhos bifurcados que afetam a integridade do jogador (ex: Corredor de Manutenção vs. Elevador de Carga).
• Interação com Inventário: Itens que podem ser usados para isolar o acesso da IA ou desferir ataques críticos.
• Ambientação Imersiva: Uso de temporizadores (u.aguarde) e manipulação de texto para simular um terminal de computador antigo.
